# Queries

## First Names
* <b>ID</b>: Auto 
* <b>VALUE</b>: A string with the name value
* <b>GENDER</b>: A char correspondent to the value.
    * ```M: Male```
    * ```F: Female```
    * ```N: Neutral```
* <b>NATIONALITY</b>: A string with a country name
